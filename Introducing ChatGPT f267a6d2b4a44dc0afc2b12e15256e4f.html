First, What is ChatGPT ? 👇

ChatGPT is an AI language model by OpenAI that generates human-like text based on input.

It can converse, answer questions, and assist with tasks like writing and problem-solving.
It was trained on vast data, and understands context and provides relevant responses.

Third, How to write text (prompt) in different way? 👇

Normally, we only write one sentence to get an answer from ChatGPT, and this method is called Zero-Shot Prompting. However, if we want better results, there are different methods that can help us write the text (prompt).

  • Provide examples with the prompt to get the results. The method is called : Few-Shot Prompting.
    Here is the example:
    Provide relevant examples ➡️ send it together with questions
    ➡️  Get ChatGPT answer.
  • Ask ChatGPT to think Step-by-Step.

    Here is the example:
    Provide questions ➡️ Ask ChatGPT to
    think Step-by-Step ➡️ Get Correct Answer.

Final Note 👇
  • Stay aware while using ChatGPT.
    1. Knowledge cutoff: ChatGPT's information is not current and has a specific cutoff date.
    1. Limitations in specialized knowledge: While knowledgeable in many areas, it may not be as accurate as domain-specific experts.
    1. Verification needed: Always double-check important information, especially for academic or professional use.
